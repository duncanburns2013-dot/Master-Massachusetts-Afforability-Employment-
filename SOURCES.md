## üìÅ Source Verification

To verify any data point:

1. **EIA Electricity Data:**
   - Visit: https://www.eia.gov/electricity/monthly/
   - Navigate to Table 5.6.A for state residential rates
   
2. **BLS Employment Data:**
   - LAUS (unemployment): https://www.bls.gov/lau/
   - CES (jobs): https://www.bls.gov/ces/
   - OES (wages): https://www.bls.gov/oes/
   
3. **Census Income Data:**
   - https://data.census.gov
   - Table S1901 for household income
   
4. **MA DPU Filings:**
   - https://eeaonline.eea.state.ma.us/DPU/Fileroom
   - Search dockets: D.P.U. 24-150, D.P.U. 25-200

5. **Housing Data:**
   - Warren Group: https://www.thewarrengroup.com/
   - Requires subscription for full data
